Sobre o PokeStatsBR

Estatísticas do Pokémon Showdown em português para a comunidade brasileira

O que é isso?

O PokeStatsBR é um site que apresenta estatísticas de batalhas do Pokémon Showdown traduzidas e adaptadas para a comunidade brasileira.

Os dados são obtidos diretamente do Smogon Stats, que analisa milhões de batalhas mensalmente para gerar estatísticas detalhadas sobre o metagame.

Funcionalidades

  • Taxa de Uso: Veja quais Pokémon são mais populares em cada formato
  • Movimentos: Descubra os moves mais usados por cada Pokémon
  • Itens: Saiba quais itens são mais efetivos
  • Habilidades: Veja as abilities mais utilizadas
  • EV Spreads: Copie as distribuições de EVs mais populares
  • Tipos Tera: Descubra os melhores Tera Types (Gen 9)
  • Parceiros de Time: Monte times sinérgicos
  • Exportar: Exporte sets prontos para o Showdown

Como Usar

  1. Escolha um formato na página inicial (VGC, OU, Doubles, etc.)
  2. Selecione o rating desejado (1760+ para top players)
  3. Clique em um Pokémon para ver detalhes completos
  4. Use o botão Exportar para copiar o set para o Showdown

Sobre os Ratings

0: Todos os jogadores (dados mais gerais)
1500: Jogadores intermediários
1630/1695: Jogadores avançados (padrão recomendado)
1760/1825: Top players (metagame de alto nível)

Atualização dos Dados

Os dados são atualizados pelo Smogon no início de cada mês, refletindo as batalhas do mês anterior. O processamento leva aproximadamente 44 horas.

Créditos

  • Smogon: smogon.com - Fonte dos dados estatísticos
  • Pokémon Showdown: pokemonshowdown.com - Simulador de batalhas
  • MunchStats: munchstats.com - Inspiração do projeto
  • Pikalytics: pikalytics.com - Referência de design
